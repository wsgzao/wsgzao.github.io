Chrome关闭硬件加速

Chrome或者Edge都可以搜索关键词关闭硬件加速提升系统稳定性

  • 使用硬件加速模式（如果可用）
  • Use hardware acceleration when available

Google Chrome 浏览器配备了「硬件加速」能力，该功能可以利用计算机的 GPU 来加速进程，以释放出更为重要的 CPU 时间。但有时，如果与计算机驱动程序不兼容，就可能会导致硬件加速功能出现异常，只能禁用它才能正常工作。

如何启用或禁用 Chrome 硬件加速，在性能和功能间取得平衡

Google Chrome comes equipped with hardware acceleration, a feature which takes advantage of your computer’s GPU to speed up processes and free vital CPU time. However, sometimes driver incompatibilities can cause this feature to misbehave and disabling it could save you a few headaches.

Chrome插件推荐

以下链接大部分都是Google Chrome Webstore，如果无法访问可以考虑从第三方下载到本地导入

Chrome Web Store

Crx4Chrome

Chrome Extension Downloader

通用

SimpleExtManager - 我个人首推的管理插件小工具，对于插件控们一定灰常有帮助

IE Tab - 为了向下兼容那些没有与时俱进的网站，大多数时候我们只能选择妥协

OneTab - 宣称节约95%内存的前提是静态网页，类似一个临时书签的记忆功能，习惯保留海量页面的朋友可以试试

Undo Closed Tabs Button - Google原生取消了恢复网页的按钮的设计，你可以通过快捷键或者历史记录恢复，但这个小工具确实可以帮到我

Tampermonkey - 老牌的油猴管理工具，和Violentmonkey暴力猴相比哪款用得顺手选择哪个

沙拉查词-聚合词典划词翻译 - 我个人觉得这是目前Chrome划词翻译插件中体验最好的

Grammarly for Chrome - 智能检查英文语法拼写错误，在Web写邮件和文档时会发现它的重要价值

Evernote Web Clipper - 中文称为印象笔记，平时会把优质的内容保存到Evernote中，虽然没有OneNote功能强大但胜在简单方便可跨平台同步

简悦 - SimpRead - 沉浸式阅读模式，还支持丰富的导出功能比如Markdown/PDF等

彩云小译 - 网页翻译插件 - 号称超越Google翻译和DeepL？

破解右键锁 - 比Enable Copy更加强大，不过大众点评的评论好像无解

Clipboard History Pro - 借助Google Chrome浏览器扩展记录剪贴板历史的小工具

爱搜资源云盘助手 - 插件方便是方便，但是可能会涉及用户隐私数据，如果介意需要慎用类似插件

Awesome screenshot and screen video - 网页截图+录屏，代替FireShot

Proxy SwitchyOmega - 轻松快捷地管理和切换多个代理设置

uBlock Origin - 类似Adblock Plus(ABP)，用于代替一款高效的网络请求过滤工具，占用极低的内存和CPU，推荐额外添加适合自己的规则效果更佳，国内用户可以安装配置更简单的广告终结者

开发者

oscnews - jaywcjlove制作的新插件，目前我的主要用途是辅助新建标签页浏览开源中国更新资讯和GitHub趋势榜，从作者TODO计划来看，Github Start管理和集成octotree部分功能都非常吸引人，加油

Remu - 通过标签分类来对 GitHub Stars 进行高效管理，借助 Gists 强大能力实现跨平台的数据同步

GitHub Repository Size - 顾名思义，可以查看 GitHub 中某个 Repo 的大小，是一个小巧实用的工具

FeHelper - WEB前端助手，All In One的一个工具，包含多个独立小应用，比如：Json工具、代码美化、代码压缩、二维码、Postman、markdown、网页油猴、便签笔记、信息加密与解密、随机密码生成、Crontab等等

Stylus - Stylus是基于Stylish调整网页外观的用户样式管理器

Octotree - 在GitHub上显示代码树

SpanTree - 在GitLab上显示代码树

ImageAssistant - 国人开发的图片助手(ImageAssistant) 批量图片下载器

Copycat - 我主要是为了复制网页直接转化为markdown

Copyfish - 免费版已经支持中文和英文的OCR在线识别，够用且准确

JSONViewer - 一个页面格式化多条 JSON，提升工作效率

SingleFile - 将完整的页面保存到一个 HTML 文件中

Unblock Youku - 我平时很少看国内在线视频，不过这款良心插件已经坚持了5年，作者不容易。如果速度不稳定大家也可以购买国内的阿里云、腾讯云等，我始终相信云即未来

为什么你们就是不能加个空格呢？ - 对文字排版有强迫症的福音吧

v2ex plus - 可能是 v2ex 最好用的扩展

一键离线下载+ - 一键自动将磁链、电驴或其他下载资源离线下载至网盘，自动将磁力链接转换为种子。

懒人专用，自用组合型多功能脚本 - 为避免争议，各位可以阅读脚本描述后决定是否使用

百度文库破解加强、 CSDN使用增强、知乎使用增强、抖音去水印下载、全网VIP视频破解，免费在线看，等多功能脚本 - 这个脚本道理同上，不多做介绍了

YAWF - 新浪微博根据关键词、作者、话题、来源等过滤微博，清理版面，以及其他改造功能
