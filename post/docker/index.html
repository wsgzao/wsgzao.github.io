Docker 常用命令

Docker Compose

Compose 是 Docker 公司推出的一个工具软件，可以管理多个 Docker 容器组成一个应用。你需要定义一个 YAML 格式的配置文件 docker-compose.yml，写好多个容器之间的调用关系。然后，只要一个命令，就能同时启动 / 关闭这些容器。

查看版本

docker-compose –version

启动所有服务

docker-compose up

关闭所有服务

docker-compose stop
