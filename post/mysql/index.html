登录参数说明

  • 大写字母P代表端口，小写字母p代表密码。
  • 输入用户名和密码时，需要在-u和-p后面直接输入，不可以有空格。
  • -p后面也可以不用输入密码，执行命令后再输入，以防止密码明文显示。

MySQL 8.0有新的安全要求，不能像之前的版本那样一次性创建用户并授权需要先创建用户，再进行授权操作
