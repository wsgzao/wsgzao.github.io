前言

相信大家更换自己笔记本电脑的内存时一定是得心应手，即便是一名新手也可以很轻松的动手实现，其实服务器的内存更换也很简单，关机->挪盖->按指定顺序插拔。不过这里有一个很重要的共性前提，需要清楚了解当前硬件所匹配的内存大小型号和插槽数量等依赖关系，这样我们在更换内存时才会更加游刃有余哈。
