前言

Markdown是一种轻量级的标记语言，流行程度已经得到了GitHub和Stack Overflow的广泛支持，作为普通人我们也可以轻松上手，结合Hexo我们可以十分优雅的分享知识。

对比2014年我写Markdown推广文章的时候，目前国内主流的站点基本都开始支持Markdown语法，从一开始的简书和SegmentFault到后起之秀的知乎和掘金等平台都在鼓励大家积极学习和使用Markdown，我相信掌握Markdown书写习惯一定会对大家有所帮助。
