By wsgzao

发表于 2016-04-19

前言

很多人都认为我是计算机高手，其实我只会帮你修理下你的电脑啦，曾几何时我也帮助过好多好多的同学和老师，尝试和体验各种各样的软件，但谁又能想到我曾今也对你们的电脑做过很多失败的坑爹操作，把你们最宝贵的数据撸得灰飞烟灭。作为一名非职业 HelpDesk，我一定不是最最优秀的那一群技术大牛，但我也努力超越了 99.99% 的从业者，Because I just love and enjoy thinking to do better and better.

如果你认同 高效，实用 的软件使用理念就跟着我走吧，让我们一起寻找 适合自己的好用软件。

长期更新此文以督促自己的懒惰，我要用佳软改变你一生的习惯。

硬件选型

选择笔记本不一定为了办公和轻便，使用台式机也未必只是为了玩游戏和大屏享受，手机和平板也可以辅助我们工作，无论选择哪种硬件，使用起来能够赏心悦目，得心应手，提高效率才是我们追求的本质。

笔记本

  1. 从颜值角度出发我会推荐 Dell XPS 和 MacBook
  2. 从实用方向考虑我会选择 ThinkPad X250 和 ThinkPad X1
  3. 其它好的选择当然也是随心啦，比如大神船和最有可能乱入的小米和华为

配件

常用软件

特色推荐

Listary - http://www.listary.com/

  • 如果 Mac 下的 Alfred 作为排名第一的效率工具类神器，那你还在犹豫什么呢，必装

阿呆猫 - http://www.admflt.com/

  • ADSafe 名气太响受到制约，移动端去广告可继续使用

Http File Server - http://www.rejetto.com/hfs/?f=dl

  • 十分方便的 HTTP 文件共享服务器

PC Hunter - http://www.xuetr.com/
